Diabetes Detection

socials

This web app uses a neural network using PyTorch to predict whether a patient has diabetes or not. The model was trained on a dataset containing 100 000 patients. The dataset was obtained from kaggle and can be found down below.

To predict diabetes, upload a CSV file containing the following columns in this order:

  • age
  • hypertension
  • heart_disease
  • bmi
  • HbA1c_level
  • blood_glucose_level
  • diabetes
